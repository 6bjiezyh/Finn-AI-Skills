# 选题灵感库字段升级说明

## 目标

- 保留原有字段与历史数据
- 支持 `daily-ai-topic-planner` 的完整结构化输出
- 支持后续自动上传飞书（按 `选题ID` upsert）

## 字段调整

保留原字段：

- 选题ID
- 选题标题
- 内容赛道
- 内容类型建议
- 目标人群
- 内容目标
- 热度来源
- 痛点描述
- 核心结果承诺
- 关键词
- 优先级
- 状态
- 计划周次
- 关联内容ID
- 备注

新增字段：

- 日期：选题生成日期（例如 `2026-02-12`）
- 来源类型：`历史` 或 `AI原创`
- 一级分类：用于固定分类配额（AI自动化 / Vibe Coding / AIGC / AI提效 / AI变现案例）
- 选题角度：对应 `topic_angle`
- 内容大纲：对应 `content_outline`
- 变现钩子：对应 `monetization_hook`
- CTA：行动召唤
- 时效性：`hot` / `evergreen`
- 难度：`low` / `medium` / `high`
- 评分：数值评分（建议 0-10）
