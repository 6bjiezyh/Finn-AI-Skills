# Hot Signal Sources

## Weighted Source Mix

Use this fixed source weight model:

- Platform content trends: 50%
- Official release updates: 30%
- Trend and demand signals: 20%

## Platform Content Trends (50%)

Track high-engagement content from:

- Bilibili (AI / automation / tool channels)
- Douyin
- Kuaishou
- Xiaohongshu
- WeChat Channels

Focus on:

- high engagement tutorials
- controversial viewpoints
- proven case studies
- repeat pain points in comments

## Official Release Updates (30%)

Track updates from:

- OpenAI
- Anthropic
- Google AI
- n8n
- 扣子/Coze
- key AI tools and changelog pages

Focus on:

- product launches
- important feature updates
- pricing/model changes
- integration changes affecting creators or SMEs

## Trend and Demand Signals (20%)

Track demand signals from:

- Baidu Index
- Google Trends
- WeChat Index
- Zhihu / comment threads / Q&A clusters

Focus on:

- repeated user questions
- rising search intent
- scenario-driven needs tied to income or efficiency

## Time Window Rules

- Primary scan window: last 24 hours.
- Secondary scan window: last 7 days.
- If 24h signals are weak, expand weight of 7d signals and backfill with evergreen topics.
